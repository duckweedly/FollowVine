FollowVine V1: GPT-Image-2 Pixel Flipbook

每页都是一张生成图；业务层负责路径、点击、缓存、分享和重试。

Core Loop

1. 输入主题

中文 topic + 受众 + 风格

例：RAG 是怎么工作的，讲给产品经理

2. 生成整页图解

gpt-image-2 生成 16:9 页面

图片本身就是学习页面

3. 点击下钻

记录 x/y 坐标 + 父图 + 路径上下文

下一页聚焦这个区域

A

纯像素 V1

最快接近 Flipbook 体验。热点不预先标注，用户点任意位置；后台用父图 + 坐标 + prompt 生成下一页。

B

像素图 + 隐性热点

图片仍由 gpt-image-2 生成，但业务层生成 3-6 个推荐点击点，用透明层辅助命中和更稳的下钻。

C

像素图 + 来源面板

中间是完整图片；右侧保留文本来源、路径和节点解释，降低“图片说错了没人发现”的风险。
