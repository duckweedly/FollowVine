# FollowVine V1 产品方案

## 一句话定位

中文 AI 视觉 Flipbook: 输入一个知识点, 生成一张可点击的中文图解页; 点图中任意位置, 继续生成下一张深入解释的图解页。

## V1 目标

验证用户是否真的愿意通过“点击图解”学习:

1. 输入中文主题。
2. 生成第一张 16:9 中文图解页。
3. 用户点击图中任意位置。
4. 生成聚焦点击区域的下一页。
5. 用户能返回、跳转历史页、分享路径。

## 不做

- 不做 SVG/HTML 语义图解层。
- 不做只点固定热点的学习图。
- 不做完整账号系统。
- 不做复杂后台。
- 不做自由 prompt 编辑。
- 不做 AI PPT、白板、笔记全家桶。

## 核心体验

```text
输入主题
→ 选择风格
→ gpt-image-2 生成 16:9 中文图解页
→ 用户点击任意位置
→ 后端用 red-circle trick 标注点击处
→ gpt-image-2 生成下一张深入图解页
→ 历史缩略图可返回/跳转
→ 路径可分享
```

## 点击机制

V1 采用“任意点击 + 后台推荐热点兜底”的思路:

- 用户体验上可以点击任意位置。
- 前端只发送归一化坐标。
- 后端用红圈标注父图点击处, 让图像模型理解点击目标。
- 后续可以补充概念吸附、推荐热点、点击热区分析, 但 V1 主路径先保持简单。

## 首发风格

| Key | 名称 | 状态 |
| --- | --- | --- |
| `watercolor_book` | 温暖水彩图解书 | 默认 |
| `chinese_science_magazine` | 中文科普杂志 | 首发 |
| `whiteboard_marker` | 白板手绘 | 首发 |
| `chalkboard` | 黑板粉笔 | hidden / experimental |

风格是固定 preset, 不允许用户直接写风格 prompt。风格 key 会进入缓存 fingerprint。
